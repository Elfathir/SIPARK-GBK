# SIPARK-GBK
SIPARK-GBK adalah aplikasi web Sistem Informasi Parkir Kendaraan 
berbasis PHP dan PostgreSQL yang mendukung pengelolaan, yakni: 
- Area parkir,
- Kendaraan (motor & mobil),
- Tarif Parkir,
- Pengguna, dan
- Transaksi Parkir.
  
Penggunaan operasi CRUD dan diimplementasi ke replikasi database Master–Slave.

Tampilan:
- Landing Page
- Login Page
- Dashboard
